# Description

**azure-repos-git-stat.sh** is used for exporting git statistics from Azure Repos. It retrieves information about commits, lines added, and lines removed for each repository within a specified date range.

## Usage

`$ ./azure-repos-git-stat.sh [azure_projects_dir]`

## Required tools/programs

`$ ./azure-repos-git-stat.sh ~/ivan/azure-projects`
